À propos de l'événement

Rejoignez-nous pour la plus grande compétition de la saison. Les finales nationales de l'ASATA Connect rassemblent les meilleurs athlètes du pays pour une journée d'excellence sportive et de networking communautaire.

Le programme inclut des sessions de qualifications le matin, suivies des phases finales l'après-midi. Une zone de rafraîchissement et de restauration sera disponible pour tous les participants accrédités.
